Etiqueta Descripcion
<address>

Define un bloque para la información de contacto.
<article>

Define un bloque de contenido autónomo que puede existir en cualquier contexto. Puede tener su propio encabezado, pie de página, secciones... Útil para una lista de publicaciones de blog.

<aside>

Define un bloque de contenido relacionado con el contenido principal. Generalmente se muestra como una barra lateral.

<header>

Define el encabezado de una página web o sección.
<div>

Define un bloque genérico de contenido, que no tiene ningún valor semántico. Se utiliza para elementos de diseño como contenedores.

<nav>

Define una sección con enlaces de navegación.
